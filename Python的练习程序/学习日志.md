##### 1.键盘录入一个数 验证这个数是否是偶数,如果是偶数, 获取其平方值否则 ,获取其立方值
```
"""
date:2020-12-19
author:王久铭
"""
a = input("Input Number:");
# input输入的默认为字符串类型
ans = int (a)
if (ans % 2 == 0) :
    ans = ans **  2;
else :
    ans = ans ** 3
print(ans)
```
##### 2.键盘录入一个百分制的成绩,90以上 等级为A,80以上  等级为B,70以上 等级为C,60以上 等级为D,其他为  E
```
"""
date:2020-12-19
author:王久铭
"""
temp = input("Input your grade：")
res = int (temp)
if (res > 90) :
    print("A")
elif (res > 80) :
    print("B")
elif (res > 70) :
    print("C")
elif (res >60) :
    print("D")
else :
    print("E")
```
